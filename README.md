Проект использует Gradle и JUnit 5.  
В современных IDE для Java, таких как Eclipse и Idea это все уже встроено, достаточно импортировать проект, нажать на классе-тесткейсе ПКМ -> Run As -> Junit test.  
Тесткейсы расположены в следующих классах:  
**src\main\java\csssr_autotest\testcase\RestTestcase.java**  
**src\main\java\csssr_autotest\testcase\SoapTestcase.java**
